# well-X-fit: Health Analytics Dashboard 🩺

A modern, lightweight, and private health analytics dashboard designed to help users track their physiological metrics in real-time. This project was developed as part of the FEE-I course at Chitkara University.

## 🚀 Live Demo
You can view the live project here: https://sussanjosee26-lang.github.io/well-X-fit/

## ✨ Features
- **Multi-Metric Calculations:** Calculate BMI, BMR (Mifflin-St Jeor), and Daily Hydration needs simultaneously.
- **Dynamic UI:** Input fields (like Age/Gender) intelligently appear or hide based on the selected health metrics.
- **Glassmorphism Design:** A futuristic, transparent interface built with modern CSS `backdrop-filter` effects.
- **Data Persistence:** Uses `localStorage` to save your calculation history locally on your browser.
- **Privacy Focused:** No data is sent to a server; all calculations happen locally on your device.

## 🛠️ Tech Stack
- **HTML5:** Semantic structure and form elements.
- **CSS3:** Custom styling, responsive layouts, and Glassmorphism effects.
- **Vanilla JavaScript:** Core logic for health formulas, DOM manipulation, and data persistence.

## 🧪 How It Works
1. **Select:** Choose the metrics you wish to calculate using the checkbox grid.
2. **Input:** Enter your Height, Weight, and Age (if required).
3. **Calculate:** Click the calculate button to see instant results.
4. **Track:** View your past results in the "Recent Calculations" log at the bottom.
